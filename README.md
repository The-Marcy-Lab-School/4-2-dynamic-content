# 4-2 Dynamic Content

Practice creating elements dynamically with JavaScript, rendering data from arrays, and using event delegation.

## Folder Structure

- **0-basic-examples/** — Practice the create-modify-append pattern
- **1-rendering-data/** — Render lists from arrays of data
- **2-event-delegation/** — Learn event propagation and delegation
- **3-card-flip-challenge/** — Build an interactive card flip game (challenge)

## Getting Started

1. Open the folder in VS Code
2. Use the Live Server extension to serve the HTML files
3. Open the browser's DevTools Console to see output
4. Follow along with the lesson and complete the TODOs in each file
